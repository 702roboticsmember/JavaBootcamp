# Do Method
A java method is like a function in other programming languages. A 'Do' Method Is just a method that runs some code given arguments (the stuff in the parenthesies).
This type of method is made using the `void` keyword, which means that this method does not return anything

Ex. 
```java
void SetSpeed(double leftSpeed, double rightSpeed) {
    Lmotor.set(leftSpeed)
    Rmotor.set(rightSpeed)
}

setSpeed(0.825)
```
