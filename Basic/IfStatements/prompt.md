# If Statements
if statments run the code between the {Brackets} if the statement between the (Parenthesies) are true.

putting else at the end will run other code if the statement is false

Ex:
```java
if(/* Statement that returns true or false */) {
    // code that does something
} else {
    // code that does something else
}
```

Comparison statements will compare two numbers and return true if the statement is true
```java
5 > 6 //false
5 < 6 //true
5 == 6 //false
```
